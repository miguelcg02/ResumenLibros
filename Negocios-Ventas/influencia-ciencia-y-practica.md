# LAS ARMAS DE LA INFLUENCIA

```
Ventaja de los atajos mentales: Tomar decisiones rápidamente es muy útil, se ahorra tiempo y energía mental.
Desventaja de los atajos mentales: Se toman decisiones equivocadas.
Defensa: Conocer los métodos de persuasión y conocer sus condiciones ideales de uso.
```

# RECIPROCIDAD

```
Aplicación: Crear en la persona un efecto de querer devolver un futuro favor, regalo o invitación. 
Defensa: Observar si el favor lleva consigo una intención secundaria.
```

# COMPROMISO Y COHERENCIA

```
Aplicación: La sociedad valora a los individuos coherentes. Una vez cree una opinión formada, es má sprobable que tome decisiones similares en el futuro.
Defensa: Separar la coherencia con el compromiso.
```

# APROBACIÓN SOCIAL

```
Aplicación: Las personas tienden a creer o confiar en las decisiones de los demás. efecto ganado.
Defensa: Desarrollar acciones legítimas, originalidad y no depender únicamente de las acciones de otras personas.
```

# SIMPATIA

```
Apliación: Se emplean los siguientes atractivos para atraer a las personas. Belleza, similitud, elogios y familiaridad.
Defensa: realizar análisis sobre lo presentado y no sobre el presentador.
```

# AUTORIDAD

```
Aplicación: Uso de simbolos para dar legitimidad a una supuesta autoridad, uso de títulos, ropa y automoviles para inlfuir.
Defensa: Preguntarse, "¿Es esta autoridad realmente un experto?", "¿Estoy seguro de que el experto es honesto?".
```

# ESCASEZ

```
Aplicación: producir un efecto de cantidad o tiempo limitado.
Defensa: saber reconocer que un producto no es necesariamente mejor solo porque falta.
```
